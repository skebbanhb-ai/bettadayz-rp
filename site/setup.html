Setup the Full Game

Run locally

npm install
cp .env.example .env
npm run local

Open http://localhost:3000. GitHub Pages hosts the static website/demo only; the multiplayer server uses Node.js, WebSockets, and SQLite, so the full backend must run locally or on a Node-capable host.

Push to GitHub

git init
git add .
git commit -m "BettaDayz RP v1"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/bettadayz-rp.git
git push -u origin main

Enable GitHub Pages

  1. Open the repository on GitHub.
  2. Go to Settings → Pages.
  3. Under Build and deployment, set Source to GitHub Actions.
  4. Push to main. The included workflow deploys the site/ folder.
